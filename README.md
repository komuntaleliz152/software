# Delicious Bites Restaurant Website

A professional restaurant website for Delicious Bites, located in Fort Portal, Uganda.

## Features

- Responsive design that works on all devices
- Modern, clean interface
- Full menu display with images and pricing
- About page with founder information
- Contact form for customer registration
- Consistent navigation across all pages

## Pages

- **index.html** - Home page with featured items and introduction
- **about.html** - About the restaurant and founder
- **menu.html** - Complete menu with all dishes
- **contactt.html** - Contact form and information

## Technologies Used

- HTML5
- CSS3 (custom styling in style/css/main.css)
- Font Awesome icons
- Responsive grid layout

## How to View

Simply open `index.html` in any modern web browser.

## Project Structure

```
├── index.html          # Home page
├── about.html          # About page
├── menu.html           # Menu page
├── contactt.html       # Contact page
├── style/
│   └── css/
│       └── main.css    # Main stylesheet
└── [image files]       # Food and restaurant images
```
